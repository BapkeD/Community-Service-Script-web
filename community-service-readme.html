This script is a community service system for FiveM servers. Staff members can assign players tasks as a form of punishment. Players must complete these tasks within a designated area on the map.

Installation

  1. Place the script files in your server's resource folder
  2. Add the following to your server.cfg: 
    ensure community_service
  3. Set up the database using the provided SQL file
  4. Configure staff identifiers in the server.lua file

Usage

Staff members can use the following command to assign tasks:

/assignservice [playerId] [numberOfTasks] [reason]

Players will be notified and teleported to the work area to complete their assigned tasks.
